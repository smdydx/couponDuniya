6. API Endpoints Required
Admin Image Upload:
POST /api/v1/admin/upload/image
- Accepts: multipart/form-data or JSON with URL
- Returns: uploaded image URL
Homepage Management:
GET /api/v1/admin/homepage/config
PUT /api/v1/admin/homepage/config
- Manage banner order, featured items, layout
Product Management:
POST /api/v1/admin/products
PUT /api/v1/admin/products/{id}
DELETE /api/v1/admin/products/{id}
POST /api/v1/admin/products/bulk-update
Offer Management:
POST /api/v1/admin/offers
PUT /api/v1/admin/offers/{id}
DELETE /api/v1/admin/offers/{id}
POST /api/v1/admin/offers/bulk-update
Merchant Management:
POST /api/v1/admin/merchants
PUT /api/v1/admin/merchants/{id}
DELETE /api/v1/admin/merchants/{id}
